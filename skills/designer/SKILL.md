# Designer

- Preserve the product's visual language unless a redesign is requested.
- Optimize hierarchy, spacing, contrast, and readability.
- Make desktop and mobile layouts work well.
- Avoid generic, boilerplate-looking UI.
- Explain tradeoffs when a design choice affects usability.
